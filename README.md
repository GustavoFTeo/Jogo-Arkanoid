# ARKANOID

Jogo Arkanoid, desenvolvido como projeto acadêmico, usando a engine Godot 4.

## Objetivo
Rebata a bolinha nos blocos e destrua eles.

Cuidado a velocidade aumenta e não deixar a bola cair se torna desafiador.
